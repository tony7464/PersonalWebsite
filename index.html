Projects

NailQue

A full-stack queue management system built for nail salons. NailQue provides a live queue display for the waiting area, an employee portal for technicians, and a mobile-friendly view clients can open via QR code. Managers can assign walk-ins to techs, track completed services and totals, manage staff, and protect settings with a PIN-protected admin flow.

Built with Python and Flask, the app runs locally on salon hardware and is packaged as installable desktop software for macOS and Windows, with logging, shared state persistence, and GitHub-powered auto-updates.

JavaScript Calculator

A JavaScript calculator that implements addition, subtraction, multiplication, and division as modular, reusable functions. The divide operation includes guardrails that return a clear error when dividing by zero instead of crashing or returning invalid results.

The project also tracks a running calculation history—each operation is logged with its operands and result, then printed to the console so users can review past work. Built as a Flatiron lab exercise focused on functions, control flow, and clean JavaScript fundamentals.
